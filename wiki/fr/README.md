# Wiki FR - SCADA Fusion/Fission

Ce wiki decrit l'exploitation du projet `SCADA-FusionReactor-MEK` pour une installation Mekanism pilotee via ComputerCraft.

## Objectif

- Superviser et piloter une installation multi-reacteurs.
- Integrer la gestion de reacteurs a fusion Mekanism.
- Operer via interfaces `Supervisor`, `Coordinator` et `Pocket`.
